## 🚀 Getting Started (3 Steps)

### Step 1: Setup (5 minutes)
```bash
# Follow DEPLOYMENT_GUIDE.md "Quick Start" section
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

### Step 2: Login (1 minute)
- Go to: http://127.0.0.1:8000/auth/login/
- Use your superuser credentials from step 1

### Step 3: Create Test Users (2 minutes)
- Follow: [HUONG_DAN_NHANH.md](HUONG_DAN_NHANH.md) section "Tài khoản mặc định"
- Or use Django shell to create test accounts
